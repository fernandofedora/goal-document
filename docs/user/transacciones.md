# Transacciones

## Registrar gastos e ingresos
- Abre **Transactions** y elige el tipo (Gasto o Ingreso).
- Completa los campos requeridos:
  - Descripción: qué es el movimiento.
  - Monto: número con dos decimales.
  - Fecha: formato `YYYY-MM-DD`.
  - Categoría: para gastos e ingresos; ayuda a ordenar tus reportes.
  - Método de pago (solo gastos): Efectivo o Tarjeta.
  - Tarjeta: si el método es Tarjeta, selecciona una tarjeta registrada.
- Guarda y verifica que aparezca en el listado.

## Editar y eliminar
- Puedes editar una transacción para corregir datos.
- También puedes eliminarla si no la necesitas.

## Filtros del historial
- Filtra por tipo (Gasto/Ingreso/Todos), método de pago, categoría, mes y año.
- Combina filtros para localizar movimientos específicos.

## Recomendaciones
- Usa categorías claras y consistentes.
- Registra la fecha real del movimiento para gráficos precisos.
