# Transacciones

## Registrar gastos/ingresos
- Abre **Transactions**.
- Usa el toggle para seleccionar `Expense` o `Income`.
- Completa `Description`, `Amount`, `Date`, `Category`.
- Para gastos: elige `Payment` (Cash/Card) y la `Card` si corresponde.
- Pulsa **Add**.

## Filtros de historial
- Tipo (Expense/Income/All), Payment, Category, Mes y Año.
- Combina filtros para encontrar movimientos específicos.
