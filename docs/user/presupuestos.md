# Presupuestos

## Crear presupuesto mensual
- En **Transactions** > **Budget**.
- Selecciona mes y año.
- Ingresa el monto total disponible para el mes y guarda.

## Cómo se calcula
- El sistema compara tus gastos del mes seleccionado contra el presupuesto.
- Verás el monto consumido, restante y el porcentaje consumido.

## Buenas prácticas
- Ajusta el presupuesto mensualmente según tu realidad.
- Si superas el presupuesto, revisa categorías para detectar excesos.
