# Tarjetas

## Añadir tarjeta
- En **Settings** > **Credit Cards**: ingresa nombre, color y últimos 4 dígitos.
- Guarda para que esté disponible al registrar gastos.

## Uso en transacciones
- Si eliges método de pago `Tarjeta`, selecciona la tarjeta correspondiente.
- El color te ayuda a identificarla rápidamente en listados y reportes.

## Consejos
- Usa nombres claros (por ejemplo: “Visa Personal”).
- Verifica los últimos 4 dígitos para distinguir tarjetas similares.
