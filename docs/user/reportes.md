# Reportes

## Visión general
- Balance: ingresos totales menos gastos totales.
- Número de transacciones registradas.

## Categorías
- Distribución de gastos por categoría (gráficos y listado).
- Usa colores de categorías para facilitar lectura.

## Ingresos vs gastos
- Serie por día para ver la evolución.

## Métodos de pago y tarjetas
- Desglose de gastos por método (efectivo/tarjeta).
- Monto total por cada tarjeta.

## Exportar a Excel
- Exporta el período seleccionado o todo.
- Incluye hojas de transacciones, totales, categorías, series y tarjetas.
