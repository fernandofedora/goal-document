# Introducción técnica

Este apartado reúne la documentación técnica del proyecto.

- Tecnologías: `React + Vite` para el frontend, `Node.js + Express` para el API, `PostgreSQL` para la base de datos.
- Repositorios:
  - Frontend: https://github.com/fernandofedora/goals-client
  - Backend: https://github.com/fernandofedora/goals-server

## Objetivos
- Estandarizar instalación, configuración y despliegue.
- Explicar arquitectura y responsabilidades de cada módulo.
- Documentar endpoints, modelos y flujos de negocio.
