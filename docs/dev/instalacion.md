# Instalación

## Requisitos
- Node.js 18+
- Python 3.x (para construir docs localmente)
- PostgreSQL 13+

## Frontend
- `npm install`
- `npm run dev`

## Backend
- Configurar `.env` (DB, JWT_SECRET)
- `npm install`
- `npm run dev`

## Documentación
- `pip install mkdocs-material`
- `mkdocs serve` en `goal-document/`
